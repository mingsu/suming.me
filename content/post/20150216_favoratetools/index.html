I prefer to use command line tools (CLI). Here is a list that I am using and enjoying.

  • OS: debian/ubuntu (95%) + windows(5%)
  • windows manager: FVWM
  • reminder: conky+remind+wyrd
  • calendar: conky+remind+web
  • shell: zsh+tmux
  • mail: mutt/getmail/msmtp
  • contactor: abook
  • document (doc): markdown (60%) + MS word (30%, under windows) + wps (10%)
  • presentation: MS powerpoint (70%, under windows) + html5 slides (30%)
  • scientific publish/thesis: latex (texlive)
  • pdf reader: okular
  • note: note
  • editor: vim
  • remote operating: ssh
  • remote desktop: remmina
  • virtual OS: virtualbox
  • database: mysql
  • file manager: ranger
  • internet expolor: firefox + uzbl
  • ftp: lftpd
  • image editor: gimp
  • music player: mocp
  • video player: vlc
  • data analysis: R/matlab/python
  • dictionary: iciba.com
  • net disk: nutstore
